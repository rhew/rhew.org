Haywood: Core software packages

Icecast

Icecast is an open source audio streaming server. Current version: 1.3.11 Web site: http://icecast.org/

LiveIce

LiveIce is a live source client for Icecast. Current version: distributed with Icecast Web site: http://star.arm.ac.uk/%7Espm/software/liveice.html

notlame

notlame is an MP3 encoder. Current version: 3.70 Web site: http://hive.me.gu.edu.au/not_lame/

sox

SoX is a sound file format converter. Current version: 10 Web site: http://www.spies.com/Sox/

mpg123

mpg123 is a real time MPEG Audio Player for Layer 1,2 and Layer3. Current version: 0.59r Web site: http://www.mpg123.de/

DECTalk

DECTalk is text to speech software. Current version: 4.6.1 Web site: http://www.forcecomputers.com/products/viewProducts.cfm?subList=8

Apache

Apache is an open source web server. Current version: 1.3.20 Web site: http://www.apache.org/

MySQL

MySQL is an open source relational database. Current version: 3.23.42 Web site: http://www.mysql.com/

Yahoo

Yahoo is a web portal. Current version: N/A Web site: http://www.yahoo.com/

lynx

lynx is a text based web browser. Current version: 2.8.3 Web site: http://lynx.browser.org/
